Nombre del Proyecto:

Juego del Héroe y el Tesoro

Descripción:

Esta aplicación web es un juego interactivo donde el jugador debe ayudar a un héroe llamado Topacio a encontrar un tesoro oculto en un tablero de 10x10. El juego utiliza un dado virtual para determinar el número de movimientos posibles en cada turno. El objetivo es encontrar el tesoro en el menor número de tiradas posibles, estableciendo un récord personal que se guarda en el almacenamiento local del navegador.

Características principales:

--Tablero de juego 10x10
--Movimiento del héroe basado en tiradas de dado
--Sistema de récord para el menor número de tiradas
--Interfaz gráfica con HTML y JavaScript

Mecánicas del juego:

El jugador introduce su nombre (mínimo 4 letras, sin números)
Se genera el tablero con el héroe en (0,0) y el tesoro en (9,9)
El jugador tira un dado virtual (1-6)
Se resaltan las casillas a las que el héroe puede moverse
El jugador selecciona una casilla resaltada para mover al héroe
El proceso se repite hasta encontrar el tesoro
Se registra el número de tiradas y se compara con el récord actual

Pasos para ejecutar el proyecto

Preparación de archivos:
Crea un archivo HTML con los elementos necesarios (tabla, botones, inputs)
Crea un archivo CSS para los estilos (no incluido en el código proporcionado)
Crea un archivo JavaScript con el código proporcionado

Ejecución:
Abre el archivo index.html en un navegador web moderno

Jugar:
Introduce tu nombre y haz clic en "Introducir nombre"
Presiona "Jugar" para iniciar la partida
Utiliza el botón "Tirar Dado" y haz clic en las casillas resaltadas para mover al héroe
